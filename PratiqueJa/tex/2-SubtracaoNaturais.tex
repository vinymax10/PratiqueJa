% ── DEFINIÇÃO ──────────────────────────────────────────────────────────────
\TW{Definição}

A \textbf{subtração} é a operação que determina a \textbf{diferença} entre dois números naturais.
Dado $a \geq b$, temos $a - b = c$ onde $c$ é o único natural tal que $b + c = a$.

Os termos da subtração têm nomes específicos:
\begin{itemize}
  \item \textbf{Minuendo} — o número do qual se subtrai (fica em cima)
  \item \textbf{Subtraendo} — o número que é subtraído (fica embaixo)
  \item \textbf{Diferença} — o resultado da subtração
\end{itemize}

\fcolorbox{laranja}{white}{\parbox{0.85\linewidth}{
\small\textbf{Atenção:} a subtração nos naturais só é possível quando o minuendo é
\textbf{maior ou igual} ao subtraendo. Caso contrário, o resultado não é um número natural.
}}

% ── SUBTRAÇÃO EM COLUNA ────────────────────────────────────────────────────
\TW{Subtração em Coluna}

Coloque o maior número em cima e o menor embaixo, alinhando os algarismos da \textbf{direita para a esquerda}.
Subtraia coluna por coluna, da direita para a esquerda.

\TM{Exemplo — $186 - 53$:}

\vspace{0.4cm}

$\begin{array}{rrrr}
&1&8&\textcolor{iris}{6}\\
-&&5&\textcolor{iris}{3}\\
\hline
&&&\textcolor{iris}{3}\\
\end{array}$
$\Rightarrow$
$\begin{array}{rrrr}
&1&\textcolor{iris}{8}&6\\
-&&\textcolor{iris}{5}&3\\
\hline
&&\textcolor{iris}{3}&3\\
\end{array}$
$\Rightarrow$
$\begin{array}{rrrr}
&\textcolor{iris}{1}&8&6\\
-&&5&3\\
\hline
&\textcolor{iris}{1}&3&3\\
\end{array}$

% ── EMPRÉSTIMO ─────────────────────────────────────────────────────────────
\TW{Empréstimo (Reagrupamento)}

Quando o dígito de cima é \textbf{menor} que o de baixo, empresta-se \textbf{1 unidade da coluna à esquerda}.
Isso acrescenta 10 ao dígito atual. O dígito emprestado fica riscado.

\TM{Exemplo — $332 - 14$:}

\vspace{0.4cm}

$\begin{array}{rrrr}
&&\textcolor{iris}{2}&\\
&3&\textcolor{babypink}{\cancel{3}}&\textcolor{iris}{12}\\
-&&1&\textcolor{iris}{4}\\
\hline
&&&\textcolor{iris}{8}\\
\end{array}$
$\Rightarrow$
$\begin{array}{rrrr}
&&\textcolor{iris}{2}&\\
&3&\textcolor{babypink}{\cancel{3}}&12\\
-&&\textcolor{iris}{1}&4\\
\hline
&&\textcolor{iris}{1}&8\\
\end{array}$
$\Rightarrow$
$\begin{array}{rrrr}
&&2&\\
&\textcolor{iris}{3}&\textcolor{babypink}{\cancel{3}}&12\\
-&&1&4\\
\hline
&\textcolor{iris}{3}&1&8\\
\end{array}$

% ── PROPRIEDADES ───────────────────────────────────────────────────────────
\TW{Propriedades da Subtração}

Ao contrário da adição, a subtração \textbf{não é comutativa} nem \textbf{associativa}:

\begin{itemize}
  \item \textbf{Elemento neutro à direita:} $a - 0 = a$ \quad Ex.: $7 - 0 = 7$
  \item \textbf{Subtração de si mesmo:} $a - a = 0$ \quad Ex.: $15 - 15 = 0$
  \item \textbf{Não comutativa:} $8 - 3 = 5$, mas $3 - 8 \notin \mathbb{N}$
  \item \textbf{Não associativa:} $(9-4)-2 = 3 \neq 7 = 9-(4-2)$
\end{itemize}

% ── PROVA REAL ─────────────────────────────────────────────────────────────
\TW{Prova Real}

A subtração é a operação \textbf{inversa} da adição. Para verificar o resultado,
some a diferença com o subtraendo — o resultado deve ser o minuendo:

\Formula{\text{Se } a - b = c, \text{ então } c + b = a}

\TM{Exemplos:}

$186 - 53 = 133$ \;\;→\;\; Prova: $133 + 53 = 186$ \checkmark

$332 - 14 = 318$ \;\;→\;\; Prova: $318 + 14 = 332$ \checkmark

% ── EMPRÉSTIMO COM ZEROS ───────────────────────────────────────────────────
\TW{Empréstimo com Zeros}

Quando a coluna vizinha é \textbf{zero}, o empréstimo \textbf{propaga-se em cascata}
até encontrar um dígito diferente de zero.

\TM{Exemplo — $500 - 237$:}

Unidades $0 < 7$: busca empréstimo nas dezenas, mas dezenas $= 0$.
Centenas cedem 1 às dezenas ($5 \to 4$, dezenas $\to 10$).
Dezenas cedem 1 às unidades ($10 \to 9$, unidades $\to 10$).

\vspace{0.4cm}

$\begin{array}{rrrr}
&4&9&\\
&\textcolor{babypink}{\cancel{5}}&\textcolor{babypink}{\cancel{10}}&10\\
-&2&3&7\\
\hline
&2&6&3\\
\end{array}$

\vspace{0.2cm}

Prova real: $263 + 237 = 500$ \checkmark

\fcolorbox{laranja}{white}{\parbox{0.85\linewidth}{
\small\textbf{Regra prática:} ao encontrar um zero, continue buscando à esquerda
até achar um dígito diferente de zero. Depois redistribua de volta para a direita,
coluna a coluna.
}}
