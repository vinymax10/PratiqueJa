\TW{Definição} 

O logaritmo de um número $a$ em uma base $b$ é o expoente $x$ ao qual a base $b$ deve ser elevada para produzir o número $a$.

Matematicamente, isso é expresso como:

\Formula{
\log_{b} a = x \iff a = b^x
}

Onde $x$ é logaritmo, $b$ é a base e $a$ é o logaritmando.

\TM{Exemplos:}

$\log_{2} 8 \Rightarrow 8 = 2^x \Rightarrow 2^3 = 2^x \Rightarrow x=3$

$\log_{3} 81 \Rightarrow 81 = 3^x \Rightarrow 3^4 = 3^x \Rightarrow x=4$

\TM{Tipos comuns de logaritmos:}

- Logaritmo natural $(\ln)$: Quando a base é o número de Euler ($\mathrm{e}  \approx 2,71828$). É denotado por $\ln b$ ou $\log_e b$.

- Logaritmo comum ($\log$): Quando a base é 10. É denotado por $\log b$ ou $\log_{10} b$.

\TM{Troca de base:}

A troca de base de logaritmos é uma propriedade fundamental que permite reescrever um logaritmo de uma base para outra. Essa propriedade é extremamente útil em diversas situações, especialmente quando se trabalha com calculadoras que geralmente só possuem logaritmos na base 10 (logaritmo comum, $\log$) e na base $\mathrm{e}$ (logaritmo natural, $\ln$).

Se você tem um logaritmo $\log_{b} a $ e deseja expressá-lo em uma nova base $c$, a fórmula é a seguinte:

\Formula{
\log_{b} a = \dfrac{\log_{c} a}{\log_{c} b}
}

A ideia por trás da fórmula é que qualquer logaritmo pode ser expresso como a razão entre dois logaritmos de uma mesma base arbitrária.

\TM{Exemplos:}

\vspace{8px} 
$\log_{8} 16 = \dfrac{\log_{2} 16}{\log_{2} 8} = \frac{4}{3}$ \vspace{8px} 

\T{Propriedades}

A seguir iremos apresentar as principais propriedades de logaritmos:

\TM{Logaritmo de um produto:}

Essa propriedade é uma das regras mais importantes dos logaritmos, conhecida como a Propriedade da Soma de Logaritmos ou Logaritmo de um Produto.

A propriedade estabelece que a soma de dois logaritmos que possuem a mesma base é igual ao logaritmo do produto dos seus logaritmandos, mantendo-se a mesma base. Matematicamente, é expressa como:

\Formula{
\log_{b} a + \log_{b} c = \log_{b} (a \cdot c)
}

Essa propriedade é crucial para simplificar e resolver expressões logarítmicas, permitindo combinar vários termos em um só. Historicamente, ela foi essencial para converter multiplicações em adições, facilitando cálculos antes da era digital.

\TM{Exemplos:}

$\log_{6} 4 + \log_{6} 9 = \log_{6} (4 \cdot 9) = \log_{6} 36 = 2$

\TM{Logaritmo de um quociente:}

Essa propriedade é outra regra fundamental dos logaritmos, conhecida como a Propriedade da Diferença de Logaritmos ou Logaritmo de um Quociente.

A propriedade estabelece que a diferença entre dois logaritmos que possuem a mesma base é igual ao logaritmo do quociente dos seus logaritmandos, mantendo-se a mesma base. Matematicamente, é expressa como:

\Formula{ 
\log_{b} a - \log_{b} c = \log_{b} \left( \dfrac{a}{c} \right)
}

Essa propriedade é vital para simplificar expressões logarítmicas com divisões e resolver equações e inequações que as contêm. Ela também facilita a transformação de divisões em subtrações, o que era útil para cálculos manuais, e permite expandir ou contrair expressões logarítmicas, sendo essencial em áreas como o cálculo.

\TM{Exemplos:}

\vspace{8px} 
$\log_{3} 54 - \log_{3} 2 = \log_{3} \left( \dfrac{54}{2} \right) = \log_{3} 27 = 3 $ \vspace{8px} 

\TM{Logaritmo de uma potência:}

Essa propriedade é uma das mais poderosas e frequentemente usadas dos logaritmos, conhecida como a Propriedade da Potência de um Logaritmo ou Regra do Expoente para Logaritmos.

A propriedade estabelece que o produto de uma constante (ou qualquer número real) por um logaritmo é igual ao logaritmo do logaritmando elevado a essa constante, mantendo-se a mesma base. Matematicamente, é expressa como:

\Formula{
c \cdot \log_{b} a = \log_{b} (a^c)
}

Essa propriedade é incrivelmente útil para simplificar expressões logarítmicas, permitindo que você mova expoentes para fora do logaritmo. Ela é essencial para resolver equações logarítmicas e exponenciais, pois permite "baixar" os expoentes ao aplicar logaritmos. Além disso, ajuda a combinar e separar logaritmos em expressões complexas e é crucial no cálculo diferencial para simplificar a derivação de funções.
    
\TM{Exemplos:}

$2 \cdot \log_{2} \sqrt{8} = \log_{2} (\sqrt{8})^2 = \log_{2} 8 = 3$

\TM{Logaritmo de uma potência de base:}

Essa propriedade é menos comum em sua formulação explícita, mas é uma derivação direta da fórmula de mudança de base de logaritmos e da propriedade da potência de um logaritmo. Ela lida com situações onde a base do logaritmo é uma potência.

A propriedade estabelece que se a base de um logaritmo é uma potência $b^c$, então o logaritmo pode ser reescrito como o inverso do expoente $\frac{1}{c}$ multiplicado pelo logaritmo do mesmo logaritmando na base original $b$. Matematicamente, é expressa como:

\Formula{
\log_{b^c} a = \dfrac{1}{c} \cdot \log_{b} a
}

\TM{Exemplos:}

\vspace{8px} 
$\log_{4} 8 = \log_{2^2} 8 = \dfrac{1}{2}  \cdot \log_{2} 8 = \dfrac{1}{2} \cdot 3 = \dfrac{3}{2}$ \vspace{8px} 

Essa propriedade é valiosa porque simplifica logaritmos com bases complexas, tornando-os mais fáceis de usar. Ela também ajuda a resolver equações e expressões logarítmicas, facilitando o isolamento de variáveis e os cálculos. Além disso, a propriedade aprofunda a compreensão das relações entre logaritmos, mostrando como eles se conectam às regras de expoentes.

\end{multicols}
