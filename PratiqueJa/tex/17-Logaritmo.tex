\TW{Definição}

O logaritmo de $a$ na base $b$ é o expoente $x$ ao qual $b$ deve ser elevada para produzir $a$. $b$ é a \textbf{base}, $a$ é o \textbf{logaritmando} e $x$ é o \textbf{logaritmo}.

\Formula{
\log_b a = x \iff a = b^x
}

\textbf{Condições de existência:} $b > 0,\; b \neq 1$ e $a > 0$. Não existe logaritmo de número negativo ou zero, nem com base 1.

\TM{Exemplos:}

$\log_2 8 \Rightarrow 8 = 2^x \Rightarrow 2^3 = 2^x \Rightarrow x = 3$

$\log_3 81 \Rightarrow 81 = 3^x \Rightarrow 3^4 = 3^x \Rightarrow x = 4$

$\log_5 25 \Rightarrow 25 = 5^x \Rightarrow 5^2 = 5^x \Rightarrow x = 2$

\TM{Tipos comuns de logaritmos:}

\textbf{Logaritmo natural} $(\ln)$: base $\mathrm{e} \approx 2{,}718$. Notação: $\ln a = \log_e a$.

\textbf{Logaritmo decimal} $(\log)$: base $10$. Notação: $\log a = \log_{10} a$.

\TM{Comportamento da função:}

$b > 1$ $\Rightarrow$ função \textbf{crescente} $(\log_2, \log_{10}, \ln)$

$0 < b < 1$ $\Rightarrow$ função \textbf{decrescente} $(\log_{1/2})$

% ──────────────────────────────────────────
\T{Logaritmos Especiais}

Três resultados decorrem diretamente da definição e aparecem em quase toda simplificação:

\Formula{\log_b 1 = 0}

pois $b^0 = 1$ para qualquer $b$ válido.

$\log_2 1 = 0$ \qquad $\log_{10} 1 = 0$

\Formula{\log_b b = 1}

pois $b^1 = b$.

$\log_2 2 = 1$ \qquad $\log_{10} 10 = 1$

\Formula{\log_b b^n = n}

Logaritmo cancela a potência de mesma base.

$\log_2 2^5 = 5$ \qquad $\log_{10} 10^3 = 3$

\Formula{b^{\log_b a} = a}

Potenciação e logaritmo de mesma base se cancelam.

$2^{\log_2 7} = 7$ \qquad $10^{\log 5} = 5$

% ──────────────────────────────────────────
\T{Mudança de Base}

Qualquer logaritmo pode ser expresso como a razão entre dois logaritmos de uma mesma base auxiliar $c$. Essencial para usar calculadoras, que só possuem $\log$ e $\ln$.

\Formula{
\log_b a = \dfrac{\log_c a}{\log_c b}
}

\TM{Exemplos:}

$\log_8 16 = \dfrac{\log_2 16}{\log_2 8} = \dfrac{4}{3}$ \hfill {\small\textit{(base auxiliar 2)}}

\vspace{6px}

$\log_2 5 = \dfrac{\log 5}{\log 2} \approx \dfrac{0{,}699}{0{,}301} \approx 2{,}322$

\TM{Consequências úteis:}

$\log_b a \cdot \log_a b = 1$ \qquad $\log_b a = \dfrac{1}{\log_a b}$

% ──────────────────────────────────────────
\T{Propriedades}

\TM{Logaritmo de um produto:}

\Formula{
\log_b a + \log_b c = \log_b (a \cdot c)
}

Converte multiplicação em adição — base histórica das tábuas de logaritmos.

$\log_6 4 + \log_6 9 = \log_6 36 = 2$

$\log_2 4 + \log_2 8 = \log_2 32 = 5$

\TM{Logaritmo de um quociente:}

\Formula{
\log_b a - \log_b c = \log_b \left(\dfrac{a}{c}\right)
}

$\log_3 54 - \log_3 2 = \log_3 27 = 3$

$\log_{10} 500 - \log_{10} 5 = \log_{10} 100 = 2$

\TM{Logaritmo de uma potência:}

\Formula{
c \cdot \log_b a = \log_b (a^c)
}

Permite mover expoentes para fora do logaritmo — essencial para equações exponenciais.

$\log_2 8^3 = 3 \cdot \log_2 8 = 3 \cdot 3 = 9$

$2 \cdot \log_2 \sqrt{8} = \log_2 (\sqrt{8})^2 = \log_2 8 = 3$

\TM{Logaritmo de uma potência de base:}

\Formula{
\log_{b^c} a = \dfrac{1}{c} \cdot \log_b a
}

$\log_4 8 = \log_{2^2} 8 = \dfrac{1}{2} \cdot \log_2 8 = \dfrac{3}{2}$

$\log_9 27 = \log_{3^2} 3^3 = \dfrac{1}{2} \cdot 3 = \dfrac{3}{2}$

% ──────────────────────────────────────────
\T{Equações Logarítmicas}

\textbf{Estratégia:} isolar o logaritmo e converter para forma exponencial, ou usar as propriedades para unir termos antes. Sempre verificar as condições de existência.

\TM{Tipo 1 — logaritmo isolado:}

$\log_2 x = 3 \Rightarrow x = 2^3 = 8$

$\log_5 x = -2 \Rightarrow x = 5^{-2} = \dfrac{1}{25}$

\TM{Tipo 2 — usando propriedades:}

$\log_2 x + \log_2 4 = 5$

$\log_2 (4x) = 5 \Rightarrow 4x = 32 \Rightarrow x = 8$

\vspace{6px}

$\log_3 x - \log_3 2 = 2$

$\log_3 \left(\dfrac{x}{2}\right) = 2 \Rightarrow \dfrac{x}{2} = 9 \Rightarrow x = 18$

\TM{Exercício combinado:}

Simplifique: $\log_2 4 + \log_2 8 - \log_2 2$

$= \log_2 (4 \cdot 8) - \log_2 2 = \log_2 \left(\dfrac{32}{2}\right) = \log_2 16 = \mathbf{4}$

% ──────────────────────────────────────────
\T{Aplicações Práticas}

Logaritmos modelam fenômenos onde a grandeza cresce em \textbf{ordens de magnitude}.

\TM{Escala Richter (sismos):}

$M = \log\left(\dfrac{A}{A_0}\right)$

Um sismo de magnitude 7 é 10× mais intenso que um de magnitude 6.

\TM{pH (química):}

$\text{pH} = -\log[\text{H}^+]$

$\text{pH} = 7 \Rightarrow [\text{H}^+] = 10^{-7}$ mol/L

\TM{Decibéis (acústica):}

$\text{dB} = 10 \cdot \log\left(\dfrac{I}{I_0}\right)$

Duplicar a intensidade sonora acrescenta apenas $\approx 3$ dB.
