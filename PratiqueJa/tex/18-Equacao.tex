% ──────────────────────────────────────────
\TW{Definição}

Uma \textbf{equação} estabelece uma igualdade entre expressões que contêm uma ou mais \textbf{incógnitas}. O valor que torna a igualdade verdadeira é chamado de \textbf{raiz} ou \textbf{solução} da equação. O \textbf{grau} é dado pelo maior expoente da incógnita.

Nas equações de \textbf{1º grau}, a incógnita aparece com expoente 1. A forma geral é:

\Formula{
ax + b = 0 \qquad (a \neq 0)
}

onde $a$ e $b$ são números reais, $a \neq 0$, e $x$ é a incógnita. A solução é $x = -\dfrac{b}{a}$.

\TM{Exemplos:}

$2x - 3 = 7$ \qquad $6x - 1 = 4$

$2 - 5x = 9$ \qquad $3x - x - 5 = 3$

% ──────────────────────────────────────────
\T{Como Resolver}

Isole a incógnita: agrupe os termos com $x$ de um lado e as constantes do outro. Ao \textbf{trocar um termo de lado}, inverta o seu operador. Ao final, \textbf{verifique} substituindo a raiz na equação original.

\TM{Regras de transposição:}

$+$ passa para o outro lado como $-$

$-$ passa para o outro lado como $+$

$\times$ passa para o outro lado como $\div$

$\div$ passa para o outro lado como $\times$

\TM{Roteiro de resolução:}

\textbf{1.} Expandir parênteses, se houver.

\textbf{2.} Transpor termos com $x$ para a esquerda e constantes para a direita.

\textbf{3.} Somar/subtrair termos semelhantes.

\textbf{4.} Dividir pelo coeficiente de $x$.

\textbf{5.} Verificar: substituir a raiz na equação original.

% ──────────────────────────────────────────
\T{Exemplos Básicos}

\TM{Exemplo 1:} $4x - 6 = 2$

$4x = 2 + 6$ \hfill {\small\textit{($-6$ passa como $+6$)}}

$4x = 8 \;\Rightarrow\; x = \dfrac{8}{4}$

$x = 2$

\textit{Verificação: $4\cdot2 - 6 = 8 - 6 = 2$ \checkmark}

\vspace{6pt}

\TM{Exemplo 2:} $3x - x - 5 = 3$

$3x - x = 3 + 5$ \hfill {\small\textit{($-5$ passa como $+5$)}}

$2x = 8 \;\Rightarrow\; x = \dfrac{8}{2}$

$x = 4$

\textit{Verificação: $3\cdot4 - 4 - 5 = 12 - 4 - 5 = 3$ \checkmark}

% ──────────────────────────────────────────
\T{Incógnita nos Dois Lados}

Quando a incógnita aparece em ambos os lados, transporte todos os termos com $x$ para a esquerda e todas as constantes para a direita antes de simplificar.

\TM{Exemplo 1:} $3x + 2 = x + 8$

$3x - x = 8 - 2$ \hfill {\small\textit{($x$ passa como $-x$; $2$ passa como $-2$)}}

$2x = 6 \;\Rightarrow\; x = 3$

\textit{Verificação: $3\cdot3+2=11$ e $3+8=11$ \checkmark}

\vspace{6pt}

\TM{Exemplo 2:} $5x - 4 = 2x + 11$

$5x - 2x = 11 + 4$

$3x = 15 \;\Rightarrow\; x = 5$

\textit{Verificação: $5\cdot5-4=21$ e $2\cdot5+11=21$ \checkmark}

% ──────────────────────────────────────────
\T{Equações com Parênteses}

Quando há parênteses, \textbf{expanda primeiro} multiplicando o fator externo por cada termo interno. Só então aplique as regras de transposição.

\TM{Exemplo 1:} $2(x + 3) = 10$

$2x + 6 = 10$ \hfill {\small\textit{(distributiva)}}

$2x = 10 - 6 = 4$

$x = 2$

\textit{Verificação: $2(2+3) = 2\cdot5 = 10$ \checkmark}

\vspace{6pt}

\TM{Exemplo 2:} $3(2x - 1) = 2(x + 4)$

$6x - 3 = 2x + 8$ \hfill {\small\textit{(distributiva nos dois lados)}}

$6x - 2x = 8 + 3$

$4x = 11$

$x = \dfrac{11}{4}$

% ──────────────────────────────────────────
\T{Equações com Frações}

Multiplique \textbf{todos os termos} pelo MMC dos denominadores para eliminar as frações e transformar a equação em uma sem denominadores.

\TM{Exemplo 1:} $\dfrac{x}{2} + 1 = 3$ \hfill {\small\textit{(MMC = 2)}}

$x + 2 = 6$ \hfill {\small\textit{(multiplicar tudo por 2)}}

$x = 4$

\textit{Verificação: $\dfrac{4}{2}+1 = 2+1 = 3$ \checkmark}

\vspace{6pt}

\TM{Exemplo 2:} $\dfrac{2x - 1}{3} = 5$ \hfill {\small\textit{(MMC = 3)}}

$2x - 1 = 15$ \hfill {\small\textit{(multiplicar tudo por 3)}}

$2x = 16$

$x = 8$

% ──────────────────────────────────────────
\T{Casos Especiais}

Após simplificar, se a incógnita \textbf{desaparecer}, analise a igualdade restante: se for \textbf{falsa}, não há solução; se for \textbf{verdadeira}, há infinitas soluções.

\TM{Sem solução — equação impossível:}

$2x + 3 = 2x + 7$

$2x - 2x = 7 - 3$

$0 = 4$ \hfill {\small\textit{(absurdo — nunca verdadeiro)}}

Não tem solução. $S = \emptyset$

\TM{Infinitas soluções — equação identidade:}

$3(x + 1) = 3x + 3$

$3x + 3 = 3x + 3$

$0 = 0$ \hfill {\small\textit{(sempre verdadeiro)}}

Infinitas soluções. $S = \mathbb{R}$

% ──────────────────────────────────────────
\T{Problema Contextualizado}

\textbf{Estratégia:} (1) definir a incógnita; (2) traduzir o enunciado para uma equação; (3) resolver; (4) interpretar a resposta no contexto do problema.

\TM{Problema 1:}

A soma de dois números consecutivos é 15. Quais são eles?

Seja $x$ o primeiro número; o próximo é $x + 1$.

$x + (x + 1) = 15$

$2x + 1 = 15 \;\Rightarrow\; 2x = 14$

$x = 7$ $\;\rightarrow\;$ os números são \textbf{7 e 8}.

\vspace{6pt}

\TM{Problema 2:}

Um número somado com 5 é igual ao dobro desse número menos 3. Qual é o número?

Seja $x$ o número.

$x + 5 = 2x - 3$

$5 + 3 = 2x - x$

$x = 8$

\textit{Verificação: $8+5=13$ e $2\cdot8-3=13$ \checkmark}

% ── INEQUAÇÕES DO 1º GRAU ──────────────────────────────────────────────────
\T{Inequações do 1º Grau}

Uma \textbf{inequação} é uma desigualdade que envolve uma incógnita. Os símbolos são
$<$, $>$, $\leq$ e $\geq$. As mesmas regras de transposição das equações valem, com
\textbf{uma exceção}: ao \textbf{multiplicar ou dividir por um número negativo}, o
sentido da desigualdade \textbf{inverte}.

\TM{Exemplo 1:} $3x - 2 > 7$

$3x > 9 \;\Rightarrow\; x > 3$

$\boxed{x \in (3, +\infty)}$

\TM{Exemplo 2:} $-2x + 1 \leq 5$

$-2x \leq 4 \;\Rightarrow\; x \geq -2$ \hfill {\small\textit{(dividiu por $-2$: inverteu)}}

$\boxed{x \in [-2, +\infty)}$

\TM{Exemplo 3:} $5 - 3x \leq 2x + 10$

$-5x \leq 5 \;\Rightarrow\; x \geq -1$ \hfill {\small\textit{(dividiu por $-5$: inverteu)}}

$\boxed{x \in [-1, +\infty)}$

\fcolorbox{laranja}{white}{\parbox{0.85\linewidth}{
\small\textbf{Notação de intervalo:}
$(a,b)$ — aberto (não inclui extremos); $[a,b]$ — fechado (inclui extremos);
$[a,b)$ e $(a,b]$ — semi-abertos.
}}
