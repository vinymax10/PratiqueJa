\TW{Definição} 

A Expressão Algébrica é uma expressão numérica que também contém letras. As letras são conhecidas como variáveis e são utilizadas para representar valores desconhecidos. Quando a Expressão Algébrica possui um único termo algébrico, ela é chamada de monômio. Quando possui mais de um, é chamada de polinômio. Um termo algébrico possui letras e números separados apenas por uma multiplicação entre eles. O número é chamado de coeficiente e as letras são chamadas de parte literal.

\TM{Exemplos de monômios:}

$2x \rightarrow 2 $ é o coeficiente e $ x $ é a parte literal

$3xy$

$54pqr$

\TM{Exemplos de polinômios:}

$2x + 3y$

$7a + 13bc$

$2t + 3y - 7z$

\T{Simplificação de Polinômio}

Primeiramente identifique e agrupe os monômios que possuem a mesma parte literal. Depois efetue a operação entre os coeficientes dos monômios com a parte literal idêntica.

\TM{Exemplos:}

$\textcolor{azure}{2x} + \textcolor{caribbeangreen}{3y} + \textcolor{azure}{8x} - \textcolor{caribbeangreen}{6y} = \textcolor{azure}{2x} + \textcolor{azure}{8x} + \textcolor{caribbeangreen}{3y} - \textcolor{caribbeangreen}{6y} = 
\textcolor{azure}{10x} - \textcolor{caribbeangreen}{ 3y}$
\vspace{0.5cm}

$\textcolor{azure}{3ab} + 7 + \textcolor{azure}{2ab} - \textcolor{darkpink}{5r} +  \textcolor{azure}{9ab}  - \textcolor{darkpink}{4r} = \textcolor{azure}{3ab} + \textcolor{azure}{2ab} +  \textcolor{azure}{9ab} - \textcolor{darkpink}{5r} - \textcolor{darkpink}{4r} + 7 = \textcolor{azure}{14ab} - \textcolor{darkpink}{9r} + 7 $
\vspace{0.5cm}

$\textcolor{darkpink}{6q} - \textcolor{caribbeangreen}{3mn} + \textcolor{darkpink}{7q} +  \textcolor{caribbeangreen}{11mn}  = \textcolor{darkpink}{6q}  + \textcolor{darkpink}{7q} - \textcolor{caribbeangreen}{3mn} +  \textcolor{caribbeangreen}{11mn}  = \textcolor{darkpink}{13q} + \textcolor{caribbeangreen}{8mn}$
