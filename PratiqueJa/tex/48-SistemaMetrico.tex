% ── COMPRIMENTO ──────────────────────────────────────────────────────────────
\TW{Comprimento}

Unidade base: \textbf{metro (m)}.

$$\text{km} \to \text{hm} \to \text{dam} \to \mathbf{m} \to \text{dm} \to \text{cm} \to \text{mm}$$

Cada passo à \textbf{direita}: $\div 10$ \qquad Cada passo à \textbf{esquerda}: $\times 10$

\TM{Conversões}

$3{,}5\,\text{km} = 3500\,\text{m}$ \hfill $(\times\,1000)$

$250\,\text{cm} = 2{,}5\,\text{m}$ \hfill $(\div\,100)$

$45\,\text{mm} = 4{,}5\,\text{cm}$ \hfill $(\div\,10)$

% ── MASSA ─────────────────────────────────────────────────────────────────────
\TW{Massa}

Unidade base no SI: \textbf{quilograma (kg)}.
Na escala métrica o grama (g) é o ponto central.

$1\,\text{t} = 1000\,\text{kg}$ \qquad $1\,\text{kg} = 1000\,\text{g}$

\TM{Conversões}

$2{,}5\,\text{kg} = 2500\,\text{g}$ \quad $350\,\text{g} = 0{,}35\,\text{kg}$

$1{,}5\,\text{t} = 1500\,\text{kg}$ \quad $200\,\text{mg} = 0{,}2\,\text{g}$

% ── CAPACIDADE ────────────────────────────────────────────────────────────────
\TW{Capacidade}

Unidade base: \textbf{litro (L)}.

$$\text{kL} \to \text{hL} \to \text{daL} \to \mathbf{L} \to \text{dL} \to \text{cL} \to \text{mL}$$

$1\,\text{L} = 1000\,\text{mL}$ \qquad $1\,\text{kL} = 1000\,\text{L}$

\TM{Conversões}

$2\,\text{L} = 2000\,\text{mL}$ \quad $500\,\text{mL} = 0{,}5\,\text{L}$

$3{,}5\,\text{kL} = 3500\,\text{L}$

% ── ÁREA E VOLUME ────────────────────────────────────────────────────────────
\TW{Área e Volume}

Para \textbf{área}: cada degrau na escala de comprimento = fator \textbf{100} (10²).

Para \textbf{volume}: cada degrau = fator \textbf{1000} (10³).

\TM{Área}

$1\,\text{m}^2 = 10\,000\,\text{cm}^2$ \quad $1\,\text{km}^2 = 10^6\,\text{m}^2$

$1\,\text{ha} = 10\,000\,\text{m}^2$ \hfill {\small(hectare — uso agrário)}

\TM{Volume}

$1\,\text{m}^3 = 10^6\,\text{cm}^3$ \quad $1\,\text{dm}^3 = 1\,\text{L}$

\fcolorbox{laranja}{white}{\parbox{0.85\linewidth}{
\small\textbf{Equivalência útil:} $1\,\text{dm}^3 = 1\,\text{L} = 1000\,\text{cm}^3$.
Liga capacidade e volume diretamente.
}}

% ── TEMPO ─────────────────────────────────────────────────────────────────────
\TW{Tempo}

O tempo \textbf{não} segue o padrão decimal — usa fatores 60 e 24:

$1\,\text{h} = 60\,\text{min}$ \qquad $1\,\text{min} = 60\,\text{s}$

$1\,\text{h} = 3600\,\text{s}$ \qquad $1\,\text{dia} = 24\,\text{h}$

\TM{Conversões}

$2{,}5\,\text{h} = 2\,\text{h}\;30\,\text{min} = 150\,\text{min}$

$90\,\text{min} = 1\,\text{h}\;30\,\text{min} = 1{,}5\,\text{h}$

$7200\,\text{s} = 120\,\text{min} = 2\,\text{h}$
