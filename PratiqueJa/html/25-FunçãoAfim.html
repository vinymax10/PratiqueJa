Função Afim

Matemática · Nível Intermediário

Definição

Função Afim (Função de 1º Grau)

Uma função afim é uma função \(f : \mathbb{R} \rightarrow \mathbb{R}\) definida por:

\[f(x) = ax + b \qquad a, b \in \mathbb{R},\; a \neq 0\]

O valor de \(a\) é o coeficiente angular da reta — representa a taxa de variação da função (inclinação). O valor de \(b\) é o coeficiente linear — é o valor da função quando \(x = 0\), ou seja, o ponto onde a reta cruza o eixo \(y\).

No plano cartesiano, a função afim é representada pela reta \(y = ax + b\).

Exemplo 1 — Avaliando a função

Dada \(f(x) = 3x + 4\), o coeficiente angular é \(a = 3\) e o coeficiente linear é \(b = 4\).

Calculando \(f(2)\):

\(f(2) = 3 \cdot 2 + 4\)

\(f(2) = 6 + 4\)

\(\boxed{f(2) = 10}\)

Coeficiente Angular

O coeficiente angular \(a\) mede a inclinação da reta. Dados dois pontos \(A = (x_A, y_A)\) e \(B = (x_B, y_B)\) pertencentes à reta, calcula-se:

\[a = \frac{\Delta y}{\Delta x} = \frac{y_B - y_A}{x_B - x_A}\]
Interpretação: se \(a > 0\) a função é crescente; se \(a < 0\) é decrescente; se \(a = 0\) é constante (reta horizontal, não é afim).
Exemplo 2 — Calculando o coeficiente angular

Dados os pontos \(A = (2, 3)\) e \(B = (3, 5)\):

\(a = \dfrac{5 - 3}{3 - 2}\)

\(a = \dfrac{2}{1}\)

\(\boxed{a = 2}\)

A função é crescente e varia 2 unidades em \(y\) para cada unidade em \(x\).

Coeficiente Linear

O coeficiente linear \(b\) é o valor de \(y\) quando \(x = 0\) — ponto de interseção da reta com o eixo \(y\). Para calculá-lo a partir de dois pontos: primeiro encontre \(a\), depois substitua as coordenadas de qualquer ponto conhecido na equação \(y = ax + b\) e isole \(b\).

\[b = y - ax\]
Exemplo 3 — Determinando a função a partir do gráfico

O gráfico abaixo passa pelos pontos \((-5,\,-4)\) e \((5,\,6)\). Determine a função afim \(f(x) = ax + b\).

Passo 1 — Coeficiente angular:

\(a = \dfrac{6 - (-4)}{5 - (-5)} = \dfrac{10}{10} = 1\)

Passo 2 — Coeficiente linear (usando o ponto \((5, 6)\)):

\(6 = 1 \cdot 5 + b\)

\(b = 6 - 5\)

\(b = 1\)

Resultado:

\[\boxed{f(x) = x + 1}\]
