Logaritmo

Definição, propriedades e mudança de base

Definição

O logaritmo de \(a\) na base \(b\) é o expoente \(x\) ao qual \(b\) deve ser elevada para produzir \(a\). \(b\) é a base, \(a\) é o logaritmando e \(x\) é o logaritmo.
\(\displaystyle \log_b a = x \;\iff\; a = b^x\)

Condições: \(b > 0,\; b \neq 1\) e \(a > 0\).

Exemplos

\(\log_2 8 \Rightarrow 8 = 2^x \Rightarrow 2^3 = 2^x\) \(\Rightarrow x = 3\)
\(\log_3 81 \Rightarrow 81 = 3^x \Rightarrow 3^4 = 3^x\) \(\Rightarrow x = 4\)

Tipos comuns

Log. natural \((\ln)\): base \(\mathrm{e} \approx 2{,}718\). Notação: \(\ln a = \log_e a\).
Log. decimal \((\log)\): base 10. Notação: \(\log a = \log_{10} a\).

Mudança de Base

Qualquer logaritmo pode ser expresso como a razão entre dois logaritmos de uma mesma base auxiliar \(c\). Essencial para usar calculadoras (que só têm \(\log\) e \(\ln\)).
\(\displaystyle \log_b a = \dfrac{\log_c a}{\log_c b}\)

Exemplo — base 2 como auxiliar

\(\log_8 16 = \dfrac{\log_2 16}{\log_2 8} = \dfrac{4}{3}\)

Usando a calculadora (base 10)

\(\log_2 5 = \dfrac{\log 5}{\log 2} = \dfrac{0{,}699}{0{,}301}\) \(\approx 2{,}322\)

Consequência útil

\(\log_b a \cdot \log_a b = 1\)
\(\log_b a = \dfrac{1}{\log_a b}\)

Propriedades dos Logaritmos

Produto — soma dos logaritmos
\(\log_b a + \log_b c = \log_b (a \cdot c)\) Converte multiplicação em adição — base histórica das tábuas de logaritmos. \(\log_6 4 + \log_6 9 = \log_6 36\) \(= 2\)
Quociente — diferença dos logaritmos
\(\log_b a - \log_b c = \log_b \!\left(\dfrac{a}{c}\right)\) Converte divisão em subtração. \(\log_3 54 - \log_3 2 = \log_3 27\) \(= 3\)
Potência — expoente desce como fator
\(c \cdot \log_b a = \log_b (a^c)\) Permite mover expoentes para fora do logaritmo. \(2 \cdot \log_2 \sqrt{8} = \log_2 (\sqrt{8})^2 = \log_2 8\) \(= 3\)
Base potência — divide pelo expoente da base
\(\log_{b^c} a = \dfrac{1}{c} \cdot \log_b a\) Simplifica logaritmos com bases compostas. \(\log_4 8 = \log_{2^2} 8 = \dfrac{1}{2} \cdot \log_2 8 = \dfrac{1}{2} \cdot 3\) \(= \dfrac{3}{2}\)
