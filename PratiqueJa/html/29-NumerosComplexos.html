Números Complexos

Matemática · Nível Avançado

Definição

Número Complexo

Um número complexo é toda expressão da forma \(z = a + bi\), onde \(a, b \in \mathbb{R}\) e \(i\) é a unidade imaginária, definida por \(i^2 = -1\).

\[z = a + bi \qquad i^2 = -1\]

\(a = \text{Re}(z)\) é a parte real e \(b = \text{Im}(z)\) é a parte imaginária de \(z\). O conjunto dos números complexos é denotado por \(\mathbb{C}\).

Exemplo

Para \(z = 3 + 4i\): \(\;\text{Re}(z) = 3\) e \(\text{Im}(z) = 4\).

Plano de Argand-Gauss

Um número complexo \(z = a + bi\) pode ser representado geometricamente como o ponto \((a, b)\) no plano complexo (plano de Argand-Gauss), onde o eixo horizontal é o eixo real e o eixo vertical é o eixo imaginário.

Operações

Soma e Subtração

Somam-se/subtraem-se as partes reais entre si e as imaginárias entre si:

\[(a+bi) \pm (c+di) = (a \pm c) + (b \pm d)\,i\]
Exemplos

\((3+4i) + (1+2i) = (3+1) + (4+2)i = 4 + 6i\)

\((5+3i) - (2+7i) = (5-2) + (3-7)i = 3 - 4i\)

Multiplicação

Usa-se a distributiva e a substituição \(i^2 = -1\):

\[(a+bi)(c+di) = (ac - bd) + (ad + bc)\,i\]
Exemplo

\((3+4i)(1+2i) = 3 + 6i + 4i + 8i^2\)

\(= 3 + 10i + 8(-1)\)

\(= -5 + 10i\)

Conjugado

Conjugado de um Número Complexo

O conjugado de \(z = a + bi\) é \(\bar{z} = a - bi\) — troca-se o sinal da parte imaginária.

\[z \cdot \bar{z} = a^2 + b^2 = |z|^2\]

O produto de um número complexo pelo seu conjugado é sempre um número real não-negativo. Essa propriedade é a base da divisão de complexos.

Divisão

Multiplica-se numerador e denominador pelo conjugado do denominador:

Exemplo

\(\dfrac{3+4i}{1+2i} = \dfrac{(3+4i)(1-2i)}{(1+2i)(1-2i)}\)

\(= \dfrac{3 - 6i + 4i - 8i^2}{1 - 4i^2} = \dfrac{3 - 2i + 8}{1 + 4}\)

\(= \dfrac{11 - 2i}{5} = \dfrac{11}{5} - \dfrac{2}{5}i\)

Módulo

O módulo de \(z = a + bi\) é a distância do ponto \((a, b)\) à origem no plano complexo:

\[|z| = \sqrt{a^2 + b^2}\]
Exemplo

Para \(z = 3 + 4i\):

\(|z| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = \boxed{5}\)

Potências de i

As potências de \(i\) se repetem em ciclos de 4:

Ciclo das Potências de i
\[i^1 = i \qquad i^2 = -1 \qquad i^3 = -i \qquad i^4 = 1\]

Para calcular \(i^n\), divide-se \(n\) por 4 e usa-se o resto: resto 0 → 1; resto 1 → \(i\); resto 2 → \(-1\); resto 3 → \(-i\).

Exemplo

\(i^{23}\): \(23 = 4 \times 5 + 3\) → resto 3 → \(i^{23} = -i\)
