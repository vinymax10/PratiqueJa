Equação do 2º Grau

Matemática · Nível Avançado

Definição

Equação do 2º Grau (Equação Quadrática)

Uma equação do 2º grau é toda equação que pode ser escrita na forma:

\[ax^2 + bx + c = 0 \qquad a, b, c \in \mathbb{R},\; a \neq 0\]

Onde \(a\) é o coeficiente do termo quadrático, \(b\) é o coeficiente do termo linear e \(c\) é o termo independente. As soluções (raízes) são os valores de \(x\) que satisfazem a igualdade.

Fórmula de Bhaskara

As raízes da equação de 2º grau são calculadas pela Fórmula de Bhaskara, que utiliza o discriminante \(\Delta\) (delta):

\[\Delta = b^2 - 4ac\]
\[x = \frac{-b \pm \sqrt{\Delta}}{2a}\]
Natureza das raízes: se \(\Delta > 0\) → duas raízes reais distintas; se \(\Delta = 0\) → uma raiz real (raiz dupla); se \(\Delta < 0\) → nenhuma raiz real.
Exemplo 1 — Bhaskara

Encontre as raízes de \(2x^2 - 7x + 3 = 0\).

Passo 1 — Calcular \(\Delta\):

\(\Delta = (-7)^2 - 4 \cdot 2 \cdot 3\)

\(\Delta = 49 - 24\)

\(\Delta = 25\)

Passo 2 — Aplicar Bhaskara:

\(x = \dfrac{-(-7) \pm \sqrt{25}}{2 \cdot 2} = \dfrac{7 \pm 5}{4}\)

\(x_1 = \dfrac{7 + 5}{4} = \dfrac{12}{4} = \boxed{3}\)

\(x_2 = \dfrac{7 - 5}{4} = \dfrac{2}{4} = \boxed{\dfrac{1}{2}}\)

Gráfico — A Parábola

A função quadrática \(f(x) = ax^2 + bx + c\) é representada por uma parábola. O sinal de \(a\) determina a concavidade:

Vértice da Parábola

O vértice \(V(x_v, y_v)\) é o ponto onde a parábola muda de sentido. Com \(a > 0\) é o ponto mínimo; com \(a < 0\) é o ponto máximo.

\[x_v = \frac{-b}{2a} \qquad\qquad y_v = \frac{-\Delta}{4a}\]
Exemplo 2 — Vértice da Parábola

Dada a equação \(x^2 - 10x + 9 = 0\), calcule o vértice.

\(a = 1,\; b = -10,\; c = 9\)

Calculando \(\Delta\):

\(\Delta = (-10)^2 - 4 \cdot 1 \cdot 9 = 100 - 36 = 64\)

Calculando o vértice:

\(x_v = \dfrac{-(-10)}{2 \cdot 1} = \dfrac{10}{2} = \boxed{5}\)

\(y_v = \dfrac{-64}{4 \cdot 1} = \dfrac{-64}{4} = \boxed{-16}\)

Vértice: \(V(5,\,-16)\) — ponto mínimo, pois \(a = 1 > 0\).

Propriedades

Propriedade 1 — Valor em x = 0

O coeficiente \(c\) é o valor da função quando \(x = 0\):

\[f(0) = a \cdot 0^2 + b \cdot 0 + c = c\]
Propriedade 2 — Caso b = 0 (equação incompleta)

Se \(b = 0\), a equação \(ax^2 + c = 0\) tem solução direta:

\[x = \pm\sqrt{\frac{-c}{a}}\]

Exemplo: \(2x^2 - 8 = 0 \Rightarrow x^2 = 4 \Rightarrow x = \pm 2\)

Propriedade 3 — Caso c = 0 (equação incompleta)

Se \(c = 0\), a equação \(ax^2 + bx = 0\) admite fatoração:

\[x(ax + b) = 0 \;\Rightarrow\; x_1 = 0 \quad\text{e}\quad x_2 = \frac{-b}{a}\]

Exemplo: \(3x^2 - 6x = 0 \Rightarrow x(3x-6) = 0 \Rightarrow x_1=0,\; x_2=2\)
