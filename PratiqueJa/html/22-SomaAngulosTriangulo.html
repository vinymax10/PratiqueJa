Soma dos Ângulos de um Triângulo

Geometria  ·  Nível Intermediário

Definição

A soma dos ângulos internos de um triângulo é sempre igual a \(180°\). Dados os ângulos internos \(\alpha\), \(\beta\) e \(\gamma\), temos:

\(\alpha + \beta + \gamma = 180°\)

Essa propriedade tem diversas aplicações práticas: resolução de problemas geométricos para encontrar ângulos desconhecidos, auxílio à construção e navegação em engenharia e arquitetura, e desenvolvimento de fórmulas e identidades trigonométricas que envolvem os ângulos internos dos triângulos.

Exemplo

Encontre o valor do ângulo \(x\) no triângulo ao lado:

\(65° + 30° + x = 180°\)

\(95° + x = 180°\)

\(x = 180° - 95°\)

\(x = 85°\)

Triângulo Isósceles

O triângulo isósceles possui dois lados com a mesma medida. O terceiro lado, de medida diferente, é chamado de base. Os dois ângulos formados pela base e os lados congruentes possuem a mesma medida.

Exemplo — Triângulo Isósceles

Encontre o valor de \(x\) no triângulo isósceles ao lado:

\(x + x + 40° = 180°\)

\(2x + 40° = 180°\)

\(2x = 180° - 40° = 140°\)

\(x = \dfrac{140°}{2}\)

\(x = 70°\)

Ângulo Externo

Em todo triângulo, qualquer ângulo externo é igual à soma dos dois ângulos internos não adjacentes a ele:

\(\text{ângulo externo} = \alpha + \beta\)
