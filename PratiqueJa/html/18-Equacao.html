Equações do 1º Grau

Definição, forma geral e resolução passo a passo

Definição

Uma equação estabelece uma igualdade entre expressões que contêm uma ou mais incógnitas — valores desconhecidos a determinar. O grau é dado pelo maior expoente da incógnita.

Nas equações de 1º grau, a incógnita aparece com expoente 1. A forma geral é:

\(\displaystyle ax + b = 0 \qquad a \neq 0\)

onde \(a\) e \(b\) são números reais, \(a \neq 0\), e \(x\) é a incógnita.

Exemplos de equações do 1º grau

\(2x - 3 = 7\)
\(6x - 1 = 4\)
\(2 - 5x = 9\)
\(3x - x - 5 = 3\)

Como Resolver

Isole a incógnita: agrupe os termos com \(x\) de um lado e as constantes do outro. Ao trocar um termo de lado, inverta o seu operador.

Regras de transposição

\(+\) passa para o outro lado como \(-\)
\(-\) passa para o outro lado como \(+\)
\(\times\) passa para o outro lado como \(\div\)
\(\div\) passa para o outro lado como \(\times\)

Roteiro de resolução

1. Transpor constantes para o lado direito.
2. Transpor termos com \(x\) para o lado esquerdo.
3. Somar/subtrair termos semelhantes.
4. Dividir ambos os lados pelo coeficiente de \(x\).

Exemplos Resolvidos

Exemplo 1 — \(4x - 6 = 2\)
\(4x - 6 = 2\) \(4x = 2 + 6\) (−6 passa para o outro lado como +6) \(4x = 8\) \(x = \dfrac{8}{4}\) (4 multiplicando passa dividindo) \(x = 2\)
Exemplo 2 — \(3x - x - 5 = 3\)
\(3x - x - 5 = 3\) \(3x - x = 3 + 5\) (−5 passa como +5) \(2x = 8\) (3x − x = 2x) \(x = \dfrac{8}{2}\) (2 multiplicando passa dividindo) \(x = 4\)
