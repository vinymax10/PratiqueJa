Semelhança entre Ângulos

Ângulos opostos pelo vértice, complementares, suplementares e relações em retas paralelas.

Def
Ângulos Opostos pelo Vértice
Congruência por interseção de retas

Quando duas retas se cruzam, formam quatro ângulos. Os ângulos que não são adjacentes entre si são chamados de ângulos opostos pelo vértice (ou ângulos verticais).

Dois ângulos opostos pelo vértice sempre possuem a mesma medida — são congruentes.

Exemplo resolvido

Na figura, dois ângulos adjacentes medem \(\alpha = 60°\) e \(\beta\). Encontre \(\beta\) e o ângulo oposto \(x\).

1
\(x = \alpha = 60°\)  (ângulos opostos pelo vértice são congruentes)
2
\(\beta + \alpha = 180°\)  (ângulos suplementares adjacentes)
3
\(\beta = 180° - 60° = \mathbf{120°}\)
90°
Ângulos Complementares
Soma igual a 90°
Dois ângulos são complementares quando a soma de suas medidas é igual a 90°.

Os ângulos não precisam ser adjacentes para serem complementares — basta que suas medidas somem 90°.

Exemplos

\(30° + 60° = \)90° → complementares ✓
\(45° + 45° = \)90° → complementares ✓
\(50° + 60° = 110° \neq 90°\) → não complementares ✗

Encontrando o complemento

Se \(\alpha = 37°\), então o complemento é: \(\beta = 90° - 37° = \)53°
180°
Ângulos Suplementares
Soma igual a 180°
Dois ângulos são suplementares quando a soma de suas medidas é igual a 180°.

Ângulos suplementares adjacentes formam uma linha reta — é por isso que a soma vale 180°.

Exemplos

\(60° + 120° = \)180° → suplementares ✓
\(90° + 90° = \)180° → suplementares ✓
\(70° + 80° = 150° \neq 180°\) → não suplementares ✗

Encontrando o suplemento

Se \(\alpha = 30°\), então o suplemento é: \(\beta = 180° - 30° = \)150°
∥
Retas Paralelas e Transversal
Ângulos formados pela reta transversal

Quando uma reta transversal corta duas retas paralelas, são formados 8 ângulos com relações especiais entre si.

Os ângulos correspondentes são iguais. Os ângulos \(\alpha\) e \(\beta\) são suplementares: \(\alpha + \beta = 180°\).

Exemplo resolvido

Com \(r \parallel s\) e transversal, um ângulo na reta \(r\) mede \(30°\). Encontre os demais.

1
\(x = 30°\)  (ângulo correspondente na reta \(s\))
2
\(y + x = 180°\)  (suplementares)
3
\(y = 180° - 30° = \mathbf{150°}\)
Alt
Ângulos Alternos
Internos e externos em retas paralelas
Quando uma transversal corta duas retas paralelas, os ângulos alternos internos são iguais entre si, e os ângulos alternos externos também são iguais entre si.

Exemplos

Alternos internos: ficam entre as paralelas, em lados opostos da transversal → iguais
Alternos externos: ficam fora das paralelas, em lados opostos da transversal → iguais
