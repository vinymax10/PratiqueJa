Subtração de Números com Mais de Um Dígito

Como fazer

Coloque o maior número em cima e o menor embaixo, alinhando os algarismos da direita para a esquerda. Depois efetue a subtração dos dígitos da coluna mais à direita e repita o processo para as demais colunas.

Empréstimo (Reagrupamento)

Quando emprestar?

Quando o dígito de cima é menor que o de baixo, pegue uma unidade emprestada da coluna à esquerda. Isso acrescenta 10 ao dígito atual, permitindo realizar a subtração.

Exemplo — 332 − 14 = 318

Propriedades da Subtração

Principais propriedades

Ao contrário da adição, a subtração não é comutativa e não é associativa.

Propriedades

Elemento neutro à direita: \(a - 0 = a\)  →  \(7 - 0 = 7\)

Subtração de si mesmo: \(a - a = 0\)  →  \(15 - 15 = 0\)

Não comutativa: \(a - b \neq b - a\)  →  \(8 - 3 = 5\), mas \(3 - 8\) não é natural

Não associativa: \((a - b) - c \neq a - (b - c)\)  →  \((9-4)-2 = 3\), mas \(9-(4-2) = 7\)

Prova Real da Subtração

Verificando o resultado

A subtração é a operação inversa da adição. Para verificar se o resultado está correto, some a diferença com o subtraendo: o resultado deve ser o minuendo.

\[\text{Se } a - b = c, \text{ então } c + b = a\]
Exemplo

Resultado encontrado: \(186 - 53 = 133\)

Prova real: \(133 + 53 = 186\) ✓

Empréstimo com Zeros

Quando o dígito da coluna vizinha é zero, não é possível emprestar diretamente. É preciso buscar o empréstimo em uma coluna mais à esquerda — o processo se propaga em cascata.

Exemplo — 500 − 237

Unidades: 0 < 7 → tentar emprestar das dezenas, mas dezenas = 0.
Solução: emprestar das centenas para as dezenas, depois das dezenas para as unidades.

5 10 10
− 2 3 7
2 6 3

Centenas 5 → cede 1 às dezenas → vira 4. Dezenas 0+10=10 → cede 1 às unidades → vira 9.

Unidades: \(10 - 7 = 3\)  |  Dezenas: \(9 - 3 = 6\)  |  Centenas: \(4 - 2 = 2\)

Resultado: \(500 - 237 = \mathbf{263}\)

Prova real: \(263 + 237 = 500\) ✓
