Política de Privacidade

Esta política de privacidade estabelece como os dados pessoais coletados do usuário são tratados pelo Pratique Já. Esta política de privacidade se aplica a todos os usuários que interagem com o site pratiqueja.com. Solicitamos a todos os usuários que leiam atentamente essa política para que tenham conhecimento das práticas empregadas no tratamento de dados pessoais. Também informamos aos usuários como acessar e atualizar seus dados pessoais.
Informações pessoais coletadas
A informação pessoal é toda informação obtida do usuário ou relacionada a ele. As informações coletadas são:
Informações cadastrais: nome, e-mail, data de nascimento, imagem de login.
Informações de acesso: data e hora de login e logout do sistema.
Resultado dos exercícios: data da realização, nota, tempo de duração.
Resultado das questões: armazenamos se o usuário acertou ou não a questão. Armazenamos somente a última resposta.
Dados de pagamento: dados do cartão de crédito ou débito (nome do titular do cartão, número do cartão e data de validade). Outros dados armazenados são número de pix, data de pagamento e valor.
Finalidade das informações pessoais
O tratamento de dados pessoais é necessário por questões legais, contratuais e para viabilizar a relação comercial entre o usuário e o Pratique Já. As informações pessoais armazenadas são necessárias para acompanhamento do aprendizado do aluno, para controle do uso da plataforma e melhorias com relação a prestação do serviço.
Compartilhamento das informações pessoais
As únicas informações pessoais compartilhadas são as informações de pagamento. Essas informações são enviadas aos prestadores de serviço de processamento de pagamentos. O tratamento desses dados de pagamento estão em conformidade com as leis, normas e os padrões de segurança. As informações de código de segurança do cartão de crédito não fica armazenada em nossos servidores.
Armazenamento das informações pessoais
As informações pessoais ficarão armazenadas pelo tempo que for necessário para atender aos propósitos legais e contratuais. Informamos que o Pratique Já implementa de forma contínua medidas de segurança da informação para preservar os dados armazenados e evitar o acesso não autorizado.
Alterações desta política de privacidade
As alterações a esta política de privacidade pode ocorrer a qualquer tempo, sem necessidade de aviso prévio. Por esse motivo, ressaltamos a importância de acessar frequentemente essa política para tomar conhecimento de qualquer mudança.
