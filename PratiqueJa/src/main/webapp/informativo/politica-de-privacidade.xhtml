Política de Privacidade

Esta política estabelece como os dados pessoais coletados dos usuários são tratados pelo Pratique Já. Aplica-se a todos os usuários que interagem com o site pratiqueja.com. Recomendamos a leitura atenta para conhecimento das práticas de tratamento de dados pessoais.

Informações Pessoais Coletadas

A informação pessoal é toda informação obtida do usuário ou relacionada a ele. As informações coletadas são:

Informações cadastrais: nome, e-mail, data de nascimento e imagem de login.
Informações de acesso: data e hora de login e logout do sistema.
Resultado dos exercícios: data da realização, nota e tempo de duração.
Resultado das questões: armazenamos se o usuário acertou ou não a questão — somente a última resposta é mantida.
Dados de pagamento: dados do cartão de crédito ou débito (nome do titular, número do cartão e data de validade), além de número de pix, data de pagamento e valor.

Finalidade das Informações Pessoais

O tratamento de dados pessoais é necessário por questões legais, contratuais e para viabilizar a relação comercial entre o usuário e o Pratique Já. As informações armazenadas são utilizadas para acompanhamento do aprendizado do aluno, controle do uso da plataforma e melhorias na prestação do serviço.

Compartilhamento das Informações Pessoais

As únicas informações pessoais compartilhadas são as de pagamento, enviadas aos prestadores de serviço de processamento de pagamentos. O tratamento desses dados está em conformidade com as leis, normas e padrões de segurança vigentes. O código de segurança do cartão de crédito não é armazenado em nossos servidores.

Armazenamento das Informações Pessoais

As informações pessoais ficarão armazenadas pelo tempo necessário para atender aos propósitos legais e contratuais. O Pratique Já implementa de forma contínua medidas de segurança da informação para preservar os dados armazenados e evitar acessos não autorizados.

Alterações desta Política de Privacidade

Alterações a esta política de privacidade podem ocorrer a qualquer tempo, sem necessidade de aviso prévio. Recomendamos o acesso frequente a esta página para tomar conhecimento de eventuais mudanças.

Como Entrar em Contato Conosco

Qualquer dúvida, crítica ou sugestão sobre como os dados pessoais estão sendo tratados, entre em contato pelo canal Fale Conosco.
