Def
Definição
O que é uma equação do 2º grau?
Uma equação do 2º grau (ou equação quadrática) é toda equação que pode ser escrita na forma \(ax^2 + bx + c = 0\), onde \(a, b, c \in \mathbb{R}\) e \(a \neq 0\).

O valor de \(a\) é o coeficiente do termo quadrático, \(b\) é o coeficiente do termo linear e \(c\) é o termo independente.

As raízes (ou soluções) da equação são os valores de \(x\) que tornam a igualdade verdadeira. Uma equação do 2º grau tem no máximo duas raízes reais.

Exemplo

Equação: \(2x^2 - 7x + 3 = 0\) \(a = 2\), \(b = -7\), \(c = 3\) É uma equação completa pois \(a \neq 0\), \(b \neq 0\) e \(c \neq 0\).
Δ
Fórmula de Bhaskara
Cálculo das raízes pelo discriminante
As raízes da equação de 2º grau são calculadas pela Fórmula de Bhaskara, que usa o discriminante \(\Delta\) (delta):
\(\Delta = b^2 - 4ac\)
\(x = \dfrac{-b \pm \sqrt{\Delta}}{2a}\)
Natureza das raízes: se \(\Delta > 0\) → duas raízes reais distintas (\(x_1 \neq x_2\)); se \(\Delta = 0\) → uma raiz real repetida (\(x_1 = x_2\)); se \(\Delta < 0\) → nenhuma raiz real (raízes complexas).

Exemplo — resolução completa

Encontre as raízes de \(2x^2 - 7x + 3 = 0\). Temos \(a=2\), \(b=-7\), \(c=3\). Passo 1 — Calcular \(\Delta\): \(\Delta = (-7)^2 - 4 \cdot 2 \cdot 3 = 49 - 24 = 25\) Passo 2 — Aplicar Bhaskara: \(x = \dfrac{-(-7) \pm \sqrt{25}}{2 \cdot 2} = \dfrac{7 \pm 5}{4}\) \(x_1 = \dfrac{7 + 5}{4} = \dfrac{12}{4} = 3\) \(x_2 = \dfrac{7 - 5}{4} = \dfrac{2}{4} = \dfrac{1}{2}\) \(\Delta = 25 > 0\) → duas raízes reais distintas: \(x_1 = 3\) e \(x_2 = \frac{1}{2}\).
Graf
Gráfico — A Parábola
Representação gráfica da função quadrática
A função quadrática \(f(x) = ax^2 + bx + c\) é representada por uma parábola. O sinal de \(a\) determina a concavidade: se \(a > 0\) a parábola abre para cima (ponto mínimo); se \(a < 0\) abre para baixo (ponto máximo).
\(\Delta > 0\) — dois pontos de cruzamento A parábola intercepta o eixo \(x\) em dois pontos distintos \(x_1\) e \(x_2\). Duas raízes reais: \(x_1 \neq x_2\)
\(\Delta = 0\) — tangente ao eixo A parábola toca o eixo \(x\) em exatamente um ponto — o vértice. Raiz dupla: \(x_1 = x_2 = \dfrac{-b}{2a}\)
\(\Delta < 0\) — não cruza o eixo A parábola fica inteiramente acima (\(a > 0\)) ou abaixo (\(a < 0\)) do eixo \(x\). Nenhuma raiz real.
O eixo de simetria da parábola é a reta vertical \(x = x_v = \dfrac{-b}{2a}\). As raízes, quando existem, são simétricas em relação a esse eixo: \(x_v = \dfrac{x_1 + x_2}{2}\).
V
Vértice da Parábola
Ponto de máximo ou mínimo da função
O vértice \(V(x_v, y_v)\) é o ponto onde a parábola muda de sentido — ponto mínimo se \(a > 0\) e ponto máximo se \(a < 0\).
\(x_v = \dfrac{-b}{2a} \qquad\qquad y_v = \dfrac{-\Delta}{4a}\)

Exemplo

Dada a equação \(x^2 - 10x + 9 = 0\), com \(a=1\), \(b=-10\), \(c=9\). Calculando \(\Delta\): \(\Delta = (-10)^2 - 4 \cdot 1 \cdot 9 = 100 - 36 = 64\) \(x_v = \dfrac{-(-10)}{2 \cdot 1} = \dfrac{10}{2} = 5\) \(y_v = \dfrac{-64}{4 \cdot 1} = \dfrac{-64}{4} = -16\) Vértice: \(V(5,\,-16)\) — ponto mínimo, pois \(a = 1 > 0\).
Valor ótimo: \(y_v\) é o valor mínimo de \(f(x)\) quando \(a > 0\), e o valor máximo quando \(a < 0\). Isso é útil para resolver problemas de otimização (menor custo, maior área, altura máxima, etc.).
±
Estudo do Sinal
Quando f(x) é positiva, nula ou negativa
Dados \(\Delta > 0\) com raízes \(x_1 < x_2\), o sinal de \(f(x) = ax^2 + bx + c\) depende do sinal de \(a\) e da posição de \(x\) em relação às raízes.
Se \(a > 0\) (concavidade ↑): \(f(x) > 0\) para \(x < x_1\) ou \(x > x_2\) \(f(x) = 0\) para \(x = x_1\) ou \(x = x_2\) \(f(x) < 0\) para \(x_1 < x < x_2\) (entre as raízes)
Se \(a < 0\) (concavidade ↓): \(f(x) < 0\) para \(x < x_1\) ou \(x > x_2\) \(f(x) = 0\) para \(x = x_1\) ou \(x = x_2\) \(f(x) > 0\) para \(x_1 < x < x_2\) (entre as raízes)

Exemplo — inequação do 2º grau

Para quais valores de \(x\) temos \(x^2 - 5x + 6 < 0\)? Raízes: \(\Delta = 25 - 24 = 1\) → \(x_1 = 2\) e \(x_2 = 3\) Como \(a = 1 > 0\), \(f(x) < 0\) entre as raízes.
Solução: \(x \in (2,\, 3)\)
S/P
Relações de Vieta
Soma e produto das raízes
Conhecidas as raízes \(x_1\) e \(x_2\), valem as seguintes relações sem necessidade de calcular Bhaskara individualmente:
\(x_1 + x_2 = \dfrac{-b}{a} \qquad\qquad x_1 \cdot x_2 = \dfrac{c}{a}\)

Exemplo

Para \(2x^2 - 7x + 3 = 0\) com \(x_1=3\) e \(x_2=\frac{1}{2}\): Soma: \(3 + \dfrac{1}{2} = \dfrac{7}{2} = \dfrac{-(-7)}{2} = \dfrac{-b}{a}\) ✓ Produto: \(3 \cdot \dfrac{1}{2} = \dfrac{3}{2} = \dfrac{c}{a}\) ✓
Construa uma equação com raízes \(x_1 = 2\) e \(x_2 = 5\). Soma \(= 7\), Produto \(= 10\) \(x^2 - 7x + 10 = 0\) Forma geral: \(x^2 - (x_1+x_2)x + x_1 x_2 = 0\).
Forma fatorada: se \(\Delta \geq 0\), a função quadrática pode ser escrita como \(f(x) = a(x - x_1)(x - x_2)\). Isso facilita identificar as raízes e o sinal de \(f(x)\) por inspeção.
Esp
Equações Incompletas
Casos especiais com b = 0 ou c = 0
Quando \(b = 0\) ou \(c = 0\), existem atalhos mais rápidos do que Bhaskara.
Caso \(b = 0\): \(ax^2 + c = 0\) Isola \(x^2\) diretamente: \(x^2 = -\dfrac{c}{a}\) \(x = \pm\sqrt{-\dfrac{c}{a}}\) (se \(-c/a \geq 0\)) Exemplo: \(2x^2 - 8 = 0 \Rightarrow x^2 = 4 \Rightarrow x = \pm 2\)
Caso \(c = 0\): \(ax^2 + bx = 0\) Fatore \(x\) em evidência: \(x(ax + b) = 0\) \(x_1 = 0 \quad\text{ou}\quad x_2 = -\dfrac{b}{a}\) Exemplo: \(3x^2 - 6x = 0\) \(x(3x-6)=0 \Rightarrow x_1=0,\; x_2=2\)
Propriedade útil: o coeficiente \(c\) é sempre o valor da função em \(x = 0\): \(f(0) = a \cdot 0^2 + b \cdot 0 + c = c\).
Prob
Problema Contextualizado
Aplicação da equação do 2º grau em situações reais
Estratégia: (1) identificar os coeficientes \(a\), \(b\) e \(c\); (2) calcular \(\Delta\); (3) aplicar Bhaskara; (4) interpretar as raízes no contexto do problema.

Problema 1 — Área de um retângulo

Um retângulo tem comprimento \(3\) cm maior que sua largura. Sabendo que a área é \(40\) cm², quais são as dimensões do retângulo?

Largura \(= x\), comprimento \(= x + 3\). Área: \(x(x+3) = 40\) \(x^2 + 3x - 40 = 0\) → \(a=1\), \(b=3\), \(c=-40\) \(\Delta = 9 + 160 = 169\) \(x = \dfrac{-3 \pm 13}{2}\) → \(x_1 = 5\) e \(x_2 = -8\) (descartado, pois \(x > 0\)) Largura \(= 5\) cm e comprimento \(= 8\) cm.

Problema 2 — Lançamento vertical

Um objeto é lançado verticalmente para cima com velocidade inicial de \(20\) m/s a partir do solo. A altura em metros é dada por \(h(t) = 20t - 5t^2\). (a) Em que instante(s) o objeto está a \(15\) m de altura? (b) Qual é a altura máxima atingida?

(a) \(20t - 5t^2 = 15 \;\Rightarrow\; t^2 - 4t + 3 = 0\) \(\Delta = 16 - 12 = 4\) → \(t = \dfrac{4 \pm 2}{2}\) → \(t_1 = 1\) s e \(t_2 = 3\) s O objeto passa por \(15\) m na subida (\(t=1\) s) e na descida (\(t=3\) s). (b) \(h(t) = -5t^2 + 20t\) → \(a = -5 < 0\) (máximo no vértice) \(t_v = \dfrac{-20}{2 \cdot (-5)} = 2\) s Altura máxima: \(h(2) = 20 \cdot 2 - 5 \cdot 4 = 40 - 20 = 20\) m
