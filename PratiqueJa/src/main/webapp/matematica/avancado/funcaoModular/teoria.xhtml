Def
Função Modular
\(f(x) = |x|\) — definida por casos
O módulo (valor absoluto) de um número real é sua distância à origem na reta: \[|x| = \begin{cases} x & \text{se } x \geq 0 \\ -x & \text{se } x < 0 \end{cases}\]
|5| = 5  ·  |-3| = 3  ·  |0| = 0
A função modular \(f: \mathbb{R} \to [0,+\infty)\) é definida por \(f(x) = |g(x)|\), onde \(g(x)\) é qualquer função real. O resultado é sempre não negativo.
Graf
Gráfico — Forma em V
Vértice no ponto mínimo; deslocamentos horizontal e vertical
O gráfico de \(f(x) = |x|\) tem forma de V com vértice na origem. A forma geral \(f(x) = |x - h| + k\) desloca o vértice para \((h,\, k)\).
Domínio: \(\mathbb{R}\).   Imagem: \([k,+\infty)\) para \(f(x) = |x-h|+k\).   A função é decrescente em \((-\infty, h]\) e crescente em \([h, +\infty)\).
Eq
Equações Modulares
\(|f(x)| = k\) — dois casos quando \(k \geq 0\)
\[|f(x)| = k \quad (k \geq 0) \;\Longrightarrow\; f(x) = k \;\text{ ou }\; f(x) = -k\] Se \(k < 0\), a equação não tem solução (\(S = \emptyset\)).
Exemplo 1 — \(|2x - 3| = 5\)
Caso 1: \(2x - 3 = 5 \Rightarrow x = 4\)
Caso 2: \(2x - 3 = -5 \Rightarrow x = -1\)
\(S = \{-1,\; 4\}\)
Exemplo 2 — \(|x^2 - 4| = 0\)
\(x^2 - 4 = 0 \Rightarrow x = \pm 2\)
\(S = \{-2,\; 2\}\)
Equação com dois módulos — \(|f(x)| = |g(x)|\): \[f(x) = g(x) \;\text{ ou }\; f(x) = -g(x)\]
Exemplo 3 — \(|x + 2| = |x - 1|\)
Caso 1: \(x + 2 = x - 1\) → impossível
Caso 2: \(x + 2 = -(x - 1) \Rightarrow 2x = -1 \Rightarrow x = -\tfrac{1}{2}\)
\(S = \left\{-\tfrac{1}{2}\right\}\)
Ineq
Inequações Modulares
Intervalo (menor que) ou união (maior que)
Caso \(|f(x)| < k\) \((k > 0)\): a distância à origem é menor que \(k\) — intervalo simétrico: \[|f(x)| < k \;\Longleftrightarrow\; -k < f(x) < k\]
Caso \(|f(x)| > k\) \((k > 0)\): a distância à origem é maior que \(k\) — união de intervalos: \[|f(x)| > k \;\Longleftrightarrow\; f(x) < -k \;\text{ ou }\; f(x) > k\]
Exemplo 4 — \(|x + 1| \leq 4\)
\(-4 \leq x + 1 \leq 4\)
\(-5 \leq x \leq 3\)
\(S = [-5,\; 3]\)
Exemplo 5 — \(|3x - 1| > 2\)
Caso 1: \(3x - 1 > 2 \Rightarrow x > 1\)
Caso 2: \(3x - 1 < -2 \Rightarrow x < -\tfrac{1}{3}\)
\(S = \left(-\infty,\; -\tfrac{1}{3}\right) \cup (1,\; +\infty)\)
Regra mnemônica: módulo menor → intervalo (and); módulo maior → união (or).
