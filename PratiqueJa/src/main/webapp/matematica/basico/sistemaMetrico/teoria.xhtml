m
Comprimento
Unidade base: metro (m)
A unidade padrão de comprimento é o metro (m). As demais unidades são múltiplos ou submúltiplos do metro, separados por fatores de 10.
km → hm → dam → m → dm → cm → mm Cada passo à direita: ÷ 10 Cada passo à esquerda: × 10

Exemplos de conversão

\(3{,}5\,\text{km} = 3500\,\text{m}\)  (× 1000) \(250\,\text{cm} = 2{,}5\,\text{m}\)  (÷ 100) \(45\,\text{mm} = 4{,}5\,\text{cm}\)  (÷ 10)
kg
Massa
Unidade base: quilograma (kg)
A unidade padrão de massa no SI é o quilograma (kg), embora na escala métrica o grama (g) seja o ponto central das conversões.
t → \(\ldots\) → kg → hg → dag → g → dg → cg → mg \(1\,\text{t} = 1000\,\text{kg}\) \(1\,\text{kg} = 1000\,\text{g}\)

Exemplos de conversão

\(2{,}5\,\text{kg} = 2500\,\text{g}\) \(350\,\text{g} = 0{,}35\,\text{kg}\) \(1{,}5\,\text{t} = 1500\,\text{kg}\)
L
Capacidade
Unidade base: litro (L)
A unidade de capacidade é o litro (L). As conversões seguem o mesmo padrão de múltiplos e submúltiplos de 10.
kL → hL → daL → L → dL → cL → mL \(1\,\text{L} = 1000\,\text{mL}\) \(1\,\text{kL} = 1000\,\text{L}\)

Exemplos de conversão

\(2\,\text{L} = 2000\,\text{mL}\) \(500\,\text{mL} = 0{,}5\,\text{L}\) \(3{,}5\,\text{kL} = 3500\,\text{L}\)
m²/m³
Área e Volume
Conversões quadráticas e cúbicas
Para área, cada passo na escala de comprimento equivale a um fator de 100 (10²). Para volume, cada passo equivale a 1000 (10³).

Área

km² → hm² → dam² → m² → dm² → cm² → mm² Cada passo: × 100 (direita→esquerda) ou ÷ 100 (esquerda→direita) \(1\,\text{m}^2 = 10\,000\,\text{cm}^2\) \(1\,\text{km}^2 = 1\,000\,000\,\text{m}^2\)

Volume

Cada passo: × 1000 ou ÷ 1000 \(1\,\text{m}^3 = 1\,000\,000\,\text{cm}^3\) \(1\,\text{dm}^3 = 1\,\text{L}\)  (equivalência com capacidade)
h
Tempo
Conversão entre horas, minutos e segundos
As unidades de tempo não seguem o padrão decimal: usam fatores fixos de 60 (entre segundos, minutos e horas) e 24 (horas no dia).
\(1\,\text{h} = 60\,\text{min}\) \(1\,\text{min} = 60\,\text{s}\) \(1\,\text{h} = 3600\,\text{s}\) \(1\,\text{dia} = 24\,\text{h}\)

Exemplos de conversão

\(2{,}5\,\text{h} = 2\,\text{h}\,30\,\text{min} = 150\,\text{min}\) \(90\,\text{min} = 1{,}5\,\text{h} = 1\,\text{h}\,30\,\text{min}\) \(7200\,\text{s} = 120\,\text{min} = 2\,\text{h}\)
