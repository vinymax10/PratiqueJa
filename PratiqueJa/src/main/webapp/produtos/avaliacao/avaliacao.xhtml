Perguntas frequentes

Como funciona a prova diferente para cada aluno?

Você monta a avaliação escolhendo assuntos e nível, e informa quantas versões quer. O sistema sorteia os exercícios e gera uma prova única para cada aluno — todas com o mesmo grau de dificuldade, mas com questões diferentes. Quem tentar colar do colega do lado vai estar respondendo outra prova.

Como é contada a cota mensal?

Cada prova individual gerada conta 1 na sua cota. Uma turma de 30 alunos com provas únicas consome 30. Assim, o plano Essencial (100/mês) dá para cerca de 3 turmas de 30, o Profissional (500/mês) para ~16 turmas e o Master (2.000/mês) para ~66 turmas.

Preciso pagar para testar?

Não. Você começa com 10 provas grátis, sem cartão de crédito. Assim dá para criar suas primeiras avaliações e conferir a qualidade antes de assinar qualquer plano.

Posso usar como lista de exercícios em vez de prova?

Sim. O mesmo gerador serve para listas de exercícios individualizadas — uma para cada aluno. É ideal para professores particulares que passam muitos exercícios de prática.

A prova já vem com gabarito?

Sim. Cada versão gerada acompanha o gabarito correspondente, então a correção fica simples mesmo com provas diferentes por aluno.

Sou professor particular, vale a pena?

Com certeza. Além das provas, você gera listas de exercícios individualizadas em poucos minutos. Conforme o número de alunos cresce, é só escolher um plano com mais cota — os planos são por volume, não por tipo de professor.

Posso trocar de plano depois?

Sim, você pode subir ou descer de plano quando quiser, conforme o volume de provas que precisa em cada período.

Atende escolas e cursinhos?

Sim. O plano Master cobre 2.000 provas por mês. Para redes maiores ou necessidades específicas, fale com a gente para um plano sob medida.
