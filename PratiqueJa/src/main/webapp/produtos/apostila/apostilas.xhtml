Perguntas frequentes

Qual apostila devo escolher?

Verifique o conteúdo programático de cada apostila e identifique aquela que mais se alinha aos assuntos que pretende aprender.

A apostila é física ou digital?

As apostilas são físicas com impressão colorida em papel A4 e capa dura, com excelente acabamento.

Como é feita a entrega?

A compra é processada pela Hotmart e o material é enviado para o endereço informado no checkout, com frete grátis.

Posso parcelar o pagamento?

Não. As apostilas são pagas à vista, em pagamento único, sem parcelamento. No checkout da Hotmart você escolhe entre as formas de pagamento disponíveis para quitar o valor de uma só vez.

O conteúdo é o mesmo para todos os exemplares?

Não, cada apostila possui um conjunto de exercícios exclusivos.
